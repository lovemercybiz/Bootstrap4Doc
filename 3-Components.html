Dismissing

Using the alert JavaScript plugin, it's possible to dismiss any alert inline. How:

  • Have the alert plugin or complied bootstrap /jS
  • If you’re building our JavaScript from source, it requires util.js. The compiled version includes this.
  • Add dismiss adn the .alert-dismissible class, which adds extra padding to the right of the alert & positions the .codebutton
  • With <button>, add the data-dismiss="alert" attribute to trigger the JS functionality and dismiss the alert
  • Use .fade & .show animate alerts while dismissing
Example:
.alert .alert-warning .alert-dismissible

.alert .alert-warning .alert-dismissible .fade .show

Need to use .fade & .code together

JavaScript behavior

Need to read up on JS and jquery?

Triggers

Enable dismissal via of alert JS

$('.alert').alert()

OR with data attributes on a button within the alert

Example:
data-dismiss="alert"

Methods

Method Description
$().alert() Makes an alert listen for click events on descendant elements which have the data-dismiss="alert" attribute. Not Not necessary when using the data-api’s auto-initialization.
$().alert('close') Close an alert by removing it from the DOM. If .fade and .show classes are present, the alert will fade out before being dismissed.
$().alert('dispose') Destroys an element's alert
Example: $(".alert").alert('close')

Events

Event Description
close.bs.alert This event fires immediately when the close instance method is called.
closed.bs.alert This event fires when the alert has been closed (will wait for CSS transitions to finish)
Example: 
      
          $('#myAlert').on('closed.bs.alert', function () {
            // do something…
          })
